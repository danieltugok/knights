# knights

## Project setup.
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Install json-server
```
sudo npm install -g json-server
```

### Start json-server
```
json-server --watch db.json
```


POSSIVEIS PROBLEMAS:
atualizar node: 
```
sudo npm install n -g
sudo n stable
```
